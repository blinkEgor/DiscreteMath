Дискретная математика. Она представляет собой математический аппарат для понимания вычислительных систем.

Математическое моделирование. Это процесс, который использует математику для решения практических задач.

Граф. Это модель, состоящая из определнного количества вершин соединённых между собой рёбрами.

Алгоритм это последовательность однозначных команд, выполнение которых влечёт решение поставленной задачи за конечное время.

Алгоритм прима может быть использован для нахождения минимального веса рёбер проходящих через все вершины графа.

Псевдокод это набор структурных элементов языка, подходящих для выражения алгоритмов.

Оператор приваивания. Он присваиват переменным значения.

Управляющий оператор. Он определяет порядок в котором будет проходить алгоритм.

Составной оператор. Это представление списка инструкция в ктором команды должны выполняться попорядку как они записаны.

Условный оператор. Он даёт возможность делать выбормежду альтернативой.

Оператор цикла. Это цикл в котором набор команд будет выполняться подходящее число раз.
